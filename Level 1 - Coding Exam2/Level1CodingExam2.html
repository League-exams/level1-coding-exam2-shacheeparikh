You are going to make a Java application that teaches young children the sounds that some animals make. It should look like the sample below:




When the user clicks on a button, the program will play the sound the animal makes.


You can use the method below to play the correct sound. Sample sounds have been supplied with the coding exercise project you cloned. 



    public void playNoise(String soundFile) {
		try {
			AudioClip sound = JApplet.newAudioClip(getClass().getResource(soundFile));
			sound.play();
			Thread.sleep(3400);
		} catch (Exception ex) {
			ex.printStackTrace();
		}
	}
